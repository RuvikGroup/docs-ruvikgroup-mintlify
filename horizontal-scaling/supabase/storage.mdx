## Storage

### Media bucket

- From you [project dashboard](https://supabase.com/dashboard), go to `Storage` and click on `New bucket`.
- Create a new `media` bucket to store public profile images.

![bucket](/images/supabase/new-media-bucket.png)

- Create a `profiles` folder in the root of the `media` bucket.

![bucket](/images/supabase/new-media-bucket-folder.png)

### Chat-Media bucket

- From you [project dashboard](https://supabase.com/dashboard), go to `Storage` and click on `New bucket`.
- Create a new `chat-media` bucket to store private user chats images.

![bucket](/images/supabase/new-chats-bucket.png)

- Create a `chats` folder in the root of the `chat-media` bucket.

![bucket](/images/supabase/new-chats-bucket-folder.png)

### Emails bucket

- From you [project dashboard](https://supabase.com/dashboard), go to `Storage` and click on `New bucket`.
- Create a new `emails` bucket to store email templates by language (e.g. `en.html`, `th.html`).

![bucket](/images/supabase/new-emails-bucket.png)

- In the root of the `emails` bucket, create 2 folders, first with your brand name (e.g. `linkedin`), second named `template` folder.
- Place your emails templates inside `template` folder.

![bucket](/images/supabase/new-emails-bucket-folder.png)
