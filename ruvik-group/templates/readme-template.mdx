## Guidelines

When using this template:

1. **Replace all placeholders** in square brackets with actual content
2. **Remove sections** that don't apply to your project
3. **Add sections** as needed for your specific project
4. **Keep it concise** but comprehensive
5. **Update regularly** as your project evolves
